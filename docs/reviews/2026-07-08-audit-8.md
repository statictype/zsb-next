#### API-1

- id: `API-1`
- file:line: `src/components/Navigation/NavLinks.tsx:38`
- axis: `prop-api`
- smell: Section-prefix active matching is exposed as `aria-current="page"`.
- evidence: `isActive()` deliberately lights section links for descendants at `src/components/Navigation/NavLinks.tsx:13`, so `/editions/2024` marks the `/editions` link active via the `pathname.startsWith(...)` branch at `src/components/Navigation/NavLinks.tsx:16`; that same broad active result is then emitted as `aria-current="page"` at `src/components/Navigation/NavLinks.tsx:38`. A repo search found no other `aria-current` navigation use outside this component, and the only external `Navigation` call site is the site layout at `src/app/(site)/layout.tsx:40`; the folder has no local test or index file beyond the four reviewed Navigation files.
- recommendation: replace `isActive` with separate exact-page and section-active results, using `aria-current="page"` only for exact href matches and a recipe-owned `data-active` state for section-prefix visual highlighting; this keeps the rendered active styling on descendant routes while correcting the public navigation semantics.

#### STRUCT-1

- evidence: Both mobile toggle buttons render `<Button variant="icon" className={s.toggle}>` at `src/components/Navigation/MobileMenu.tsx:27` and `src/components/Navigation/MobileMenu.tsx:49`; the shared icon button recipe sets `width: '44px'`, `height: '44px'`, `background: 'transparent'`, `borderWidth: '0'`, and transition behavior at `src/design-system/recipes/button.ts:60`, while `s.toggle` sets `width: 'touch'`, `height: 'touch'`, `background: 'transparent'`, `border: 'none'`, `padding: '0'`, and its own transition-bearing pseudo-element at `src/components/Navigation/Navigation.recipe.ts:143`. - recommendation: merge the nav toggle's 48px icon-button geometry into the shared Button recipe as a named icon size or compound variant, then leave `s.toggle` to fixed positioning, color state, and the pseudo-mark only; this shrinks duplicated button reset styles and preserves the intended touch target.
